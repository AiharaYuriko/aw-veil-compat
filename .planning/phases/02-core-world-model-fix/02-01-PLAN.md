<objective>
Inject AW shader uniforms into Veil-compiled vertex shader programs.

Purpose: When Veil compiles its own shader programs (via ShaderManager.readShader() -> DirectShaderCompiler), AW's vanilla uniform injection (via ShaderVanillaMixin wrapping ShaderInstance's ResourceProvider) is bypassed. The resulting GL programs lack AW's 7 uniforms (aw_ModelViewMatrix, etc.), causing glGetUniformLocation() to return -1 and AW's VBO uniform writes to silently fail. This fix intercepts Veil's shader source loading and applies AW's preprocessing transformations, making AW uniforms AVAILABLE in the compiled program so AW's existing "borrow current program" render pattern works correctly.

Output:
- AwShaderUniformInjector.java -- self-contained GLSL transformer (mirrors AW's ShaderPreprocessor with vanilla profile, 6 attribute mappings + 7 uniform declarations + aw_main_pre() injection). Thread-safe, stateless, zero AW-class dependency.
- VeilShaderResourceMixin.java -- @WrapOperation on ResourceProvider.getResourceOrThrow() inside ShaderManager.readShader(), applies injector to .vsh files
- Updated awveilcompat.veil.mixins.json registering the new mixin
</objective>

    Create `AwShaderUniformInjector.java` in the `com.example.awveilcompat.shader` package.

    This class is a self-contained, thread-safe, stateless GLSL vertex shader transformer that reproduces AW's ShaderPreprocessor "vanilla" profile logic. It must NOT import any AW classes (zero compile-time dependency) and must NOT reference ModConfig, Minecraft.getInstance(), or any thread-unsafe singleton.

    CRITICAL RULES:
    - The transformation MUST be PURELY ADDITIVE to the shader source. Original attribute declarations (in vec3 Position, in vec2 UV0, etc.) must remain unchanged -- the injector adds aw_Prefixed variables and uniform declarations alongside them.
    - The method MUST be idempotent: if the source already contains "aw_MatrixFlags", return it unchanged.
    - The transformation must handle both `in` and `attribute` qualifiers for vertex inputs (GLSL ES uses `in`, GLSL compatibility uses `attribute`).
    - All attribute renames must ONLY apply the `\bVARIABLE_NAME\b` regex to variable references in the shader body (step 2 of AW's register function), NOT to the attribute declaration itself (which is restored by step 3).
    - Do NOT use java.util.regex.Matcher or Pattern directly -- use String.replaceAll() like AW does, or use a simpler StringBuilder-based approach.
    - Thread safety: zero mutable state in the class, no constructor parameters, no instance fields. Use a static method `inject(String source): String`.

    Implementation specification:

    1. Check for idempotency: if `source == null` or `source.contains("aw_MatrixFlags")`, return `source` immediately.

    2. Define the 6 attribute mappings matching AW's "vanilla" profile exactly:

       | AW Name | Type | Vanilla Name | Matrix Type | Matrix Uniform | Expression |
       |---------|------|-------------|-------------|----------------|-----------|
       | aw_UV0 | vec2 | UV0 | mat4 | aw_TextureMatrix | vec2($2 * vec4($1, 1, 1)) |
       | aw_UV1 | ivec2 | UV1 | mat4 | aw_OverlayTextureMatrix | ivec2($2 * vec4($1, 1, 1)) |
       | aw_UV2 | ivec2 | UV2 | mat4 | aw_LightmapTextureMatrix | ivec2($2 * vec4($1, 1, 1)) |
       | aw_Color | vec4 | Color | vec4 | aw_ColorModulator | ($2 * $1) |
       | aw_Normal | vec3 | Normal | mat3 | aw_NormalMatrix | ($2 * $1) |
       | aw_Position | vec3 | Position | mat4 | aw_ModelViewMatrix | vec3($2 * vec4($1, 1)) |

    3. For each attribute mapping, apply AW's 3-step register logic:
       a. Find `(QUALIFIER\s+TYPE\s+)(\bVANILLA_NAME\b)(.*?;)` and replace with `$1__aw_VANILLA_NAME_aw__$3`
          - QUALIFIER is `in` or `attribute` (test both)
          - TYPE is the attribute type (vec2, ivec2, vec3, vec4)
       b. Replace `\bVANILLA_NAME\b` with `AW_NAME` in the entire source
          - This renames all variable references from e.g. `Position` to `aw_Position` in the shader body
          - Use a careful regex that matches whole words only (\b anchors)
       c. Find `(QUALIFIER\s+TYPE\s+)(\b__aw_VANILLA_NAME_aw__\b)(.*?;)` and replace with:
          `uniform MATRIX_TYPE MATRIX_UNIFORM;\nAW_TYPE AW_NAME;\n$1VANILLA_NAME$3`
          - This restores the original attribute declaration AND adds the uniform + local variable

    4. After processing all 6 attributes, check if ANY transformations were applied:
       - If no attributes were found (source may not be an AW-compatible shader), inject ONLY the 7 uniform declarations after the #version line (if present) or at the top of the file. This ensures compatibility with Veil-native shaders (pinwheel format) that don't use vanilla attribute names.
       - If attributes were found, proceed to step 5.

    5. Generate the `aw_main_pre()` function:
       ```
       #ifdef GL_ES
       uniform int aw_MatrixFlags;
       #else
       uniform int aw_MatrixFlags = 0;
       #endif

       void aw_main_pre() {
         if ((aw_MatrixFlags & 0x01) != 0) {
           aw_UV0 = aw_TextureMatrix * vec4(UV0, 1, 1);
           aw_UV1 = ivec2(aw_OverlayTextureMatrix * vec4(UV1, 1, 1));
           aw_UV2 = ivec2(aw_LightmapTextureMatrix * vec4(UV2, 1, 1));
           aw_Color = aw_ColorModulator * Color;
           aw_Normal = aw_NormalMatrix * Normal;
           aw_Position = vec3(aw_ModelViewMatrix * vec4(Position, 1));

           // Normal normalization when non-uniform scaled
           if ((aw_MatrixFlags & 0x02) != 0) {
             aw_Normal = normalize(aw_Normal);
           }
         } else {
           aw_UV0 = UV0;
           aw_UV1 = UV1;
           aw_UV2 = UV2;
           aw_Color = Color;
           aw_Normal = Normal;
           aw_Position = Position;
         }
       }
       ```

    6. Inject the `aw_main_pre()` function definition BEFORE the main() function, then inject `aw_main_pre();` as the first statement inside `main()`:
       - Find `void\s+main\s*\(\s*\)\s*\{` (with optional whitespace)
       - Insert the aw_main_pre function before main's opening, then the call after `{`:
         ```
         aw_main_pre_body
         
         void main() {
         aw_main_pre();
         ```
       - Use AW's exact pattern: `source.replaceAll("(void\\s+main\\s*\\(\\)\\s*\\{)(\\s*)", awMainPre + "\n$1$2aw_main_pre();$2$2")`

    7. For the "uniforms-only fallback" (step 4 branch): inject these after #version directive:
       ```
       uniform mat4 aw_ModelViewMatrix;
       uniform mat4 aw_TextureMatrix;
       uniform mat4 aw_OverlayTextureMatrix;
       uniform mat4 aw_LightmapTextureMatrix;
       uniform mat3 aw_NormalMatrix;
       uniform vec4 aw_ColorModulator;
       uniform int aw_MatrixFlags;
       ```

    Implementation specification:

    1. Define the mixin class:
    ```java
    @Pseudo
    @Mixin(targets = "foundry.veil.api.client.render.shader.ShaderManager")
    public class VeilShaderResourceMixin {
    
        @WrapOperation(
            method = "readShader",
            at = @At(value = "INVOKE", 
                     target = "Lnet/minecraft/server/packs/resources/ResourceProvider;getResourceOrThrow(Lnet/minecraft/resources/ResourceLocation;)Lnet/minecraft/server/packs/resources/Resource;"),
            remap = false
        )
        private Resource aw2$wrapShaderSource(ResourceProvider provider, ResourceLocation location, Operation<Resource> original) throws IOException {
            // Only intercept vertex shader files
            if (!location.getPath().endsWith(".vsh")) {
                return original.call(provider, location);
            }
            
            // Get the original resource
            Resource resource = original.call(provider, location);
            
            // Read the original shader source
            String originalSource;
            try (Reader reader = resource.openAsReader()) {
                originalSource = IOUtils.toString(reader);
            }
            
            // Transform through our injector
            String transformedSource = AwShaderUniformInjector.inject(originalSource);
            
            // If nothing changed, return original resource (avoids unnecessary wrapping)
            if (transformedSource == originalSource || transformedSource.equals(originalSource)) {
                return resource;
            }
            
            // Return a new Resource wrapping the transformed source
            byte[] transformedBytes = transformedSource.getBytes(StandardCharsets.UTF_8);
            return new Resource(
                resource.source(),
                () -> new ByteArrayInputStream(transformedBytes),
                () -> new StringReader(transformedSource)
            );
        }
    }
    ```
